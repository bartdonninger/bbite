KNSB

juni 2019 - augustus 2019

Full stack ontwikkelaar

In opdracht van Code38
X2Connect Web Api

De Koninklijke Nederlandse Schaats Bond is in Nederland verantwoordelijk voor alles wat met schaatsen te maken heeft. De KNSB is bezig met het plaatsen van extra X2 transponders op ijsbanen door Nederland heen en schaatsverenigingen kunnen de informatie van deze transponders op vragen.

Om dit mogelijk te maken heb ik in samenwerking met Code38 de X2Connect Web Api opgezet. Dit is een .NET Core Web Api waar clients, in dit geval schaatsverenigingen, gegevens op kunnen vragen die zijn geregistreerd door de X2 transponders. Het authentiseren van de clients gebeurt via het OAuth 2.0 protocol. Tevens is er een user interface gemaakt waarmee de KNSB zelf nieuwe clients toe kan voegen om toegang tot de X2Connect Web Api te verlenen.

Full stack ontwikkelaar

In dienst van Rubicon
Web platform

PGGM&CO is een ledenclub voor iedereen die werkt of heeft gewerkt in de zorg en welzijn. PGGM&CO helpt werkgevers en werknemers bij het creëren van gezonde arbeidsorganisaties. Zij streven naar een gezonder arbeidsklimaat, zodat werknemers vitaal en met plezier aan het werk blijven. Zij bereiken hun leden via hun website die een neutrale en betrouwbare plek is vol met tips, artikelen en hulpmiddelen.

Bij PGGM&CO werk ik in een klein scrumteam dat verantwoordelijk is voor het web platform. Binnen dit team heb ik een rol als ontwikkelaar. Het team is verantwoordelijk voor alle ICT faciliteiten die te maken hebben met de website. Dit gaat van het neer zetten van een OTAP straat tot aan het inrichten van de databases en het beheren van de infrastructuur.

De website is in 2018 omgebouwd naar een nieuwe versie en draait o.a. op moderne technieken zoals AngularJs, .NET Core Web Api’s, Umbraco, Azure Functions. Ik ben in mijn rol als ontwikkelaar met al deze technieken bezig geweest. Naast het technische gedeelte help ik ook mee om wensen vanuit de business zo concreet mogelijk te maken en te kijken hoe deze binnen het ICT landschap van PGGM&CO passen.

Zorginzicht

Het project Zorginzicht gaat voor een portaal zorgen waarop samengewerkt kan worden aan de informatievoorziening van kwaliteits- en informatiestandaarden binnen de zorg.

Binnen het project Zorginzicht heb ik de rol van ontwikkelaar. Samen met de klant en mijn teamgenoten ontwerp en ontwikkel ik aan SharePoint 2013, wat de basis vormt voor het portaal van Zorginzicht. Ik verricht hier zowel frontend als backend werkzaamheden.
